Regras CSS

Uma regra CSS é aquela que contém seletores, propriedades e valores que quando aplicado a um elemento html tem a capacidade de alterar seu estilo. Uma regra css sempre estará dentro de chaves e possuirá um seletor. Por exemplo: h1{propriedade:valor;}

Os seletores são divididos em três tipos basicos: seletore de tipo, de id e de classe. Cada um desses seletores tem uma especificidade que determina sua abrangência.

Exemplo

Seletor de Tipo

Esse seletor é caracterizado por se assemelhar a uma tag html. Ele sempre será representado pela letra da tag. Por examplo: h1{propriedade:valor}

Seletor de Id

Os seletores de id são aqueles que são aplicados aplicados elementos html com um id especifico. Esse seletor começa pelo simbolo #. Por exemplo: #Div1{propriedade:valor}

Seletor de Classe

Os Seletores de classe são aqueles em que atribuimos ou aplicamos estilos css a elementos html que possuam o atributo class, esse seletor pode ser aplicado a mais de um elemento html. Esse seletor sempre começara por um "ponto" "." seguido do nome da classe. Por exemplo: .classe1{propriedade:valor}
