O que é o Grid layout

O Grid layout é bidimensional, permitindo que o desenvolvedor possa separar o layout da pagina em linhas e colunas, diferentemente do flex box que é unidimensional, ordenando os elementos em linha.

Flexgrid ou Grid Layout

O flexbox e grid layout podem ser utilziados em conjunto. voce pode aplicar o flexbox dentro de um layout que estaja usando o gridlayout, aproveitando o melhor das duas tecnicas

A escolha entre flexbox e grid layout não se trata de um substituir o outro, mas sim de entender as necessidades especificasde cada layout, aplicando as suas necessidades.
